# Image_to_ascii_art
*This is a continuation/rewriting of image_to_ascii_art_c but in javascript so that I can later add a gui and turn it into a webapp/website

The purpose of this project is for me to learn the web development languages by using them for the first time. This program converts an uploaded image into ASCII art by sampling pixels at certain intervals and mapping a corrosponding symbol to it based on it's brightness. 

There are two outputs. The left one is based on drawing the symbols with a canvas and the right one is a purely text output that can be copy and pasted.

Features that I am figuring out:
    Text based:
        better font size scaling
        an option for color
        edge detection
    
    Canvas based:
        higher resolution
        toggling color
        having control over the size of the characters
